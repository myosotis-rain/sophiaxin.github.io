The Wellesley Web: TechConnect

About the Project

The Wellesley Web is a pre-professional networking platform specifically designed for Wellesley College students and alumni in tech and tech-adjacent fields. Our team developed the website's front-end and back-end entirely from scratch.

Technologies: Python, HTML, CSS, SQL, JavaScript
Team (Alphabetical Order): Claire Cao, Diya Khanna, Chloe Shim, Sophia Xin

My Contribution

As the lead frontend designer, I implemented the majority of the visual designs and CSS styling while contributing to both backend and frontend core functionalities. This project marked my journey from having no prior knowledge in HTML and CSS to independently completing comprehensive front-end development through dedicated self-learning.
